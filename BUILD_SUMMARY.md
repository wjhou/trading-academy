## 🚀 How to Use It

### Option 1: Read as Markdown (Simplest)
```bash
cd /Users/houwenjun/Desktop/Projects/trading-academy
cat book/introduction.md
cat book/module-01/lesson-01-what-is-trading.md
```

### Option 2: Build Interactive Book (Recommended)
```bash
# Install mdBook (one-time)
brew install mdbook

# Serve the book
cd trading-academy
mdbook serve --open
# Opens at http://localhost:3000
```

### Option 3: Deploy to GitHub Pages
```bash
# Push to GitHub
git remote add origin https://github.com/YOUR_USERNAME/trading-academy.git
git push -u origin main

# Enable GitHub Pages in repo settings
# Book will be live at: https://YOUR_USERNAME.github.io/trading-academy
```
