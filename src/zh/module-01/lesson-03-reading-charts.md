## 📊 什么是价格数据？

每个交易周期（分钟、小时、日）都有四个关键价格：

### OHLC - 四个价格

**开盘价（Open）**：周期开始时的第一个价格
**最高价（High）**：周期内的最高价格
**最低价（Low）**：周期内的最低价格
**收盘价（Close）**：周期结束时的最后价格

### 示例：AAPL 的日线 OHLC

```
日期：2024-01-15
开盘价：$150.00  (上午 9:30)
最高价：$153.50  (下午 2:15)
最低价：$149.25  (上午 10:00)
收盘价：$152.75  (下午 4:00)
```

**这告诉我们什么**：
- 股票以 $150 开盘
- 达到 $153.50 的高点（买家推高价格）
- 一度跌至 $149.25（卖家压低价格）
- 当天以 $152.75 收盘（净涨 $2.75）

## 🕯️ 蜡烛图

蜡烛图以直观的方式可视化 OHLC 数据。

### 蜡烛图的结构

```
        |  <- 上影线（最高价）
        |
    ┌───────┐
    │       │  <- 实体（开盘价到收盘价）
    │       │
    └───────┘
        |
        |  <- 下影线（最低价）
```

### 阳线（价格上涨）

```
收盘价：$155  |
         ┌───────┐
         │ 绿色  │  实体 = 收盘价 > 开盘价
         │       │
         └───────┘
开盘价：$150   |
```

- **实体**：开盘价到收盘价（填充绿色/白色）
- **上影线**：收盘价之上的最高价
- **下影线**：开盘价之下的最低价
- **含义**：买家获胜，价格上涨

### 阴线（价格下跌）

```
开盘价：$155   |
         ┌───────┐
         │ 红色  │  实体 = 收盘价 < 开盘价
         │       │
         └───────┘
收盘价：$150  |
```

- **实体**：开盘价到收盘价（填充红色/黑色）
- **上影线**：开盘价之上的最高价
- **下影线**：收盘价之下的最低价
- **含义**：卖家获胜，价格下跌

## 📈 解读蜡烛图形态

### 长实体 = 强劲走势

```
    |
┌───────┐
│       │  <- 长绿色实体
│       │     强劲买盘压力
│       │
└───────┘
    |
```

### 短实体 = 犹豫不决

```
    |
    |
  ┌─┐  <- 短实体
  └─┘     买卖双方势均力敌
    |
    |
```

### 长影线 = 拒绝

```
    |
    |  <- 长上影线
  ┌─┐     价格在高位被拒绝
  └─┘
    |  <- 长下影线
    |     价格在低位被拒绝
```

## 🔍 常见蜡烛图形态

### 1. 十字星（Doji）- 犹豫不决

```
    |
    |
    +  <- 开盘价 = 收盘价
    |
    |
```

**含义**：买卖双方势均力敌，可能反转

### 2. 锤子线（Hammer）- 看涨反转

```

  ┌─┐
  └─┘
    |
    |  <- 长下影线
    |
```

**含义**：卖家压低价格，买家强力反击

### 3. 流星线（Shooting Star）- 看跌反转

```
    |
    |  <- 长上影线
    |
  ┌─┐
  └─┘
```

**含义**：买家推高价格，卖家强力反击

### 4. 吞没形态（Engulfing Pattern）

**看涨吞没**：
```
  ┌─┐ ┌───────┐
  │红│ │ 绿色  │  <- 绿色蜡烛吞没红色
  └─┘ │       │
      └───────┘
```

**看跌吞没**：
```
┌───────┐ ┌─┐
│ 绿色  │ │红│  <- 红色蜡烛吞没绿色
│       │ └─┘
└───────┘
```

## ⏰ 时间周期

同一只股票在不同时间周期上看起来完全不同！

### 常见时间周期

- **1 分钟**：每根蜡烛 = 1 分钟（日内交易者）
- **5 分钟**：每根蜡烛 = 5 分钟（超短线交易者）
- **15 分钟**：每根蜡烛 = 15 分钟（日内交易）
- **1 小时**：每根蜡烛 = 1 小时（波段交易者）
- **日线**：每根蜡烛 = 1 天（持仓交易者）
- **周线**：每根蜡烛 = 1 周（长期投资）

### 示例：AAPL 在不同时间周期上

**1 分钟图**：显示每个微小波动，非常嘈杂
**日线图**：显示整体趋势，过滤噪音
**周线图**：显示长期方向，非常平滑

**规则**：时间周期越高 = 信号越可靠

## 💻 实践操作：使用 stock-agent-system 可视化

让我们生成一个带蜡烛图的价格图表！

```python
# 创建脚本来可视化价格数据
cd /Users/houwenjun/Desktop/Projects/stock-agent-system
source .venv/bin/activate

python3 << 'EOF'
import asyncio
from datetime import datetime, timedelta
from stock_agent.core.config import load_config
from stock_agent.data.aggregator import DataAggregator
from stock_agent.core.types import TimeFrame, Market

async def main():
    config = load_config("configs/default.yaml")
    aggregator = DataAggregator.from_config(config)

    # 获取 30 天的日线数据
    symbol = "AAPL"
    end = datetime.now()
    start = end - timedelta(days=30)

    bars = await aggregator.get_historical_bars(
        symbol=symbol,
        timeframe=TimeFrame.DAILY,
        start=start,
        end=end,
        market=Market.US,
    )

    print(f"\n{symbol} - 最近 10 天：")
    print("-" * 80)
    print(f"{'日期':<12} {'开盘价':>8} {'最高价':>8} {'最低价':>8} {'收盘价':>8} {'成交量':>12}")
    print("-" * 80)

    for bar in bars[-10:]:
        date_str = bar.timestamp.strftime("%Y-%m-%d")
        print(f"{date_str:<12} {bar.open:>8.2f} {bar.high:>8.2f} "
              f"{bar.low:>8.2f} {bar.close:>8.2f} {bar.volume:>12,.0f}")

    await aggregator.close()

asyncio.run(main())
EOF
```

## 🎯 阅读图表：分步指南

### 步骤 1：识别时间周期
- 日线？小时线？5 分钟线？
- 这决定了每根蜡烛代表什么

### 步骤 2：观察整体趋势
- 价格总体上是上涨、下跌还是横盘？
- 先不要关注单个蜡烛

### 步骤 3：检查单个蜡烛
- 长实体 = 强劲走势
- 长影线 = 拒绝/犹豫
- 颜色 = 谁赢了（买家还是卖家）

### 步骤 4：寻找形态
- 顶部/底部的十字星
- 吞没形态
- 锤子线和流星线

### 步骤 5：考虑成交量
- 高成交量 = 强烈信念
- 低成交量 = 疲弱走势，可能反转

## 🔑 关键要点

1. **OHLC** = 开盘价、最高价、最低价、收盘价（四个关键价格）
2. **蜡烛图**直观地可视化 OHLC 数据
3. **绿色/白色** = 看涨（收盘价 > 开盘价）
4. **红色/黑色** = 看跌（收盘价 < 开盘价）
5. **实体大小** = 走势强度
6. **影线** = 拒绝和犹豫
7. **时间周期很重要** = 时间周期越高越可靠
8. **形态**提供未来方向的线索
