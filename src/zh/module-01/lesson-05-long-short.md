## 📈 做多（买入）

**做多** = 买入股票，期待其价格上涨。

### 运作方式

```
1. 以 $150 买入 AAPL
2. 价格上涨至 $160
3. 以 $160 卖出
4. 利润：每股 $10
```

### 交易示例

```
资金：$10,000
买入：100 股 AAPL，价格 $150 = $15,000（使用保证金）
价格上涨至 $160
卖出：100 股，价格 $160 = $16,000
利润：$1,000（6.7% 回报率）
```

### 特点

✅ **上涨空间无限**：股票可以无限上涨
✅ **下跌风险有限**：最多只会损失投资金额
✅ **简单**：容易理解
✅ **常见**：大多数人只做多

## 📉 做空（卖出）

**做空** = 卖出你并不拥有的股票，期待其价格下跌。

### 运作方式

```
1. 从券商借入 100 股 AAPL
2. 以 $150 卖出借来的股票 = 收到 $15,000
3. 价格下跌至 $140
4. 以 $140 买回 100 股 = 花费 $14,000
5. 将股票归还给券商
6. 利润：$1,000
```

### 可视化解释

```
正常（做多）：
低价买入 → 高价卖出 → 获利

做空：
高价卖出 → 低价买入 → 获利
（只是反过来了！）
```

### 做空交易示例

```
1. 以 $200 做空 100 股 TSLA
   - 借入并卖出：收到 $20,000
2. 价格下跌至 $180
3. 以 $180 买回 100 股
   - 成本：$18,000
4. 将股票归还给券商
5. 利润：$2,000
```

### 特点

❌ **上涨空间有限**：股票跌至 $0 时利润最大
❌ **下跌风险无限**：股票可以无限上涨
❌ **复杂**：需要保证金账户
❌ **高风险**：可能损失超过投资金额
❌ **成本高**：需要支付借券费用

### 卖空的危险性

**场景**：你以 $200 做空 TSLA

```
如果价格跌至 $0：
利润：每股 $200（100% 收益）

如果价格涨至 $400：
损失：每股 $200（100% 亏损）

如果价格涨至 $600：
损失：每股 $400（200% 亏损！）

如果价格涨至 $1000：
损失：每股 $800（400% 亏损！！）
```

**风险**：损失可能超过你的初始投资！

## 🐂 多头与空头

### 多头（乐观派）

**多头** = 认为价格会上涨的人

特点：
- 买入股票（做多）
- 预期经济增长
- 积极情绪
- "市场会上涨！"

**牛市 (Bull Market)**：
- 价格从低点上涨 20% 以上
- 经济强劲
- 信心高涨
- 例子：2009-2020 年（最长牛市）

### 空头（悲观派）

**空头** = 认为价格会下跌的人

特点：
- 卖出股票或做空
- 预期经济衰退
- 消极情绪
- "市场会崩盘！"

**熊市 (Bear Market)**：
- 价格从高点下跌 20% 以上
- 经济疲软
- 信心低迷
- 例子：2008 年金融危机、2020 年新冠疫情崩盘

### 为什么叫"牛"和"熊"？

**牛 (Bull)**：攻击时用牛角向上顶 ↑
**熊 (Bear)**：攻击时用熊掌向下拍 ↓

## 📊 市场情绪

### 情绪指标

**看涨情绪**：
- 交易量大
- 股票创新高
- 波动率低（VIX）
- 正面新闻报道

**看跌情绪**：
- 恐慌性抛售
- 股票创新低
- 波动率高（VIX 飙升）
- 负面新闻报道

### 逆向思维

**当所有人都看涨时** → 市场可能见顶
**当所有人都看跌时** → 市场可能见底

名言："在别人贪婪时恐惧，在别人恐惧时贪婪" —— 沃伦·巴菲特 (Warren Buffett)

## 🎯 做多与做空：何时使用

### 何时做多：

✅ 你相信股票会上涨
✅ 牛市或上升趋势
✅ 基本面积极
✅ 你是初学者（更安全）
✅ 长期投资

### 何时做空：

✅ 你相信股票会下跌
✅ 熊市或下降趋势
✅ 基本面消极
✅ 你有经验（风险更高）
✅ 短期交易

### 给初学者：坚持做多

**原因**：
- 风险有限（不会损失超过投资金额）
- 更容易理解
- 没有借券成本
- 从历史上看，市场长期趋势向上

## 🔑 关键要点

1. **做多** = 低买高卖（从上涨中获利）
2. **做空** = 高卖低买（从下跌中获利）
3. **做多风险** = 有限（最多损失投资金额）
4. **做空风险** = 无限（可能损失超过投资金额）
5. **多头** = 乐观，预期价格上涨
6. **空头** = 悲观，预期价格下跌
7. **牛市** = 价格上涨 20% 以上（好时光）
8. **熊市** = 价格下跌 20% 以上（坏时光）
9. **给初学者** = 坚持做多头寸
10. **市场趋势** = 长期向上（历史规律）

## ⚠️ 重要警告

### 关于卖空

- **风险极高**：可能损失超过投资金额
- **需要经验**：不适合初学者
- **成本高昂**：借券费用会累积
- **时机关键**：可能判断正确但时机过早
- **轧空风险**：如果做空者众多，价格可能飙升

### 著名的轧空案例

**游戏驿站 (GameStop, GME) 2021 年**：
- 股票被大量做空
- 散户投资者大量买入
- 空头被迫买回平仓
- 价格从 $20 涨至 $480
- 做空者损失数十亿美元
