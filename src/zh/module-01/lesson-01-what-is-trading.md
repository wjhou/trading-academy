## 📖 什么是交易？

**交易**是买卖金融工具（如股票、债券、货币）的行为，目的是从价格波动中获利。

### 交易 vs 投资

| 交易 | 投资 |
|------|------|
| 短期到中期（几天到几个月） | 长期（几年到几十年） |
| 从价格波动中获利 | 从公司增长 + 股息中获利 |
| 需要主动管理 | 被动，买入并持有 |
| 风险更高，潜在回报更高 | 风险较低，稳定回报 |
| 专注技术分析 | 专注基本面分析 |

**示例**：
- **投资者**：以150美元买入苹果股票，持有10年，以300美元卖出
- **交易员**：以150美元买入苹果，下周以155美元卖出，多次重复

## 📈 什么是股票？

**股票**（也称为股份或权益）代表对公司的所有权。

### 公司为什么发行股票

1. **筹集资金**：在不贷款的情况下获得资金来发展业务
2. **流动性**：创始人和早期投资者可以出售他们的股份
3. **员工薪酬**：提供股票期权来吸引人才
4. **公开估值**：为公司建立市场价格

### 股票类型

- **普通股**：有投票权，股息（如果公司支付），公司倒闭时排在最后
- **优先股**：无投票权，固定股息，优先于普通股

### 股票代码（Ticker）

每只股票都有唯一标识符：
- **AAPL** = 苹果公司
- **MSFT** = 微软公司
- **GOOGL** = Alphabet Inc.（谷歌）
- **TSLA** = 特斯拉公司

## 🏛️ 股票市场和交易所

**股票交易所**是买卖股票的市场。

### 主要美国交易所

1. **NYSE（纽约证券交易所）**
   - 最古老、最大的交易所
   - 传统"场内交易" + 电子交易
   - 蓝筹股公司（IBM、可口可乐等）

2. **NASDAQ（纳斯达克）**
   - 完全电子化交易所
   - 科技股为主（苹果、微软、亚马逊）
   - 执行速度更快

3. **其他交易所**
   - AMEX、BATS、IEX等

### 交易所如何运作

```
买家想要 AAPL @ $150  ←→  交易所  ←→  卖家有 AAPL @ $150
                                ↓
                          交易执行！
                                ↓
                    买家获得股票，卖家获得资金
```

### 交易时间

- **常规时间**：美东时间上午9:30 - 下午4:00（周一至周五）
- **盘前交易**：美东时间上午4:00 - 上午9:30
- **盘后交易**：美东时间下午4:00 - 晚上8:00

**注意**：大部分交易发生在常规时间。盘前/盘后交易量较低，价差较大。

## 🤝 经纪商的作用

你不能直接访问交易所 - 你需要一个**经纪商**。

### 经纪商做什么

1. **执行订单**：将你的买卖订单发送到交易所
2. **持有你的资产**：在账户中保管你的股票和现金
3. **提供工具**：图表、研究、分析工具
4. **保证金/杠杆**：借钱给你交易（高级功能）

### 经纪商类型

- **全方位服务**：个人建议、研究、高费用（如摩根士丹利）
- **折扣经纪**：自主交易、低费用（如Alpaca、盈透证券）
- **机器人顾问**：自动化投资（如Betterment、Wealthfront）

- **Alpaca**：美股，模拟和实盘交易
- **模拟经纪商**：无需真钱练习
- **QMT/easytrader**：中国A股

## 💰 交易如何赚钱

### 基本概念

**低买高卖**

```
以$150买入AAPL  →  价格涨到$160  →  以$160卖出  →  利润：每股$10
```

### 交易示例

```
1. 你的账户有$10,000
2. 以$150买入100股AAPL = 花费$15,000（使用保证金）
3. AAPL涨到$155
4. 以$155卖出100股 = 收到$15,500
5. 利润：$500（减去费用）
6. 回报率：一次交易3.3%
```

### 现实情况

- 并非每笔交易都盈利
- 费用和税收减少利润
- 时机很难把握
- 情绪可能导致错误决策
- 风险管理至关重要

## 🔑 关键概念

### 买价和卖价

- **买价（Bid）**：买家愿意支付的最高价格
- **卖价（Ask）**：卖家愿意接受的最低价格
- **价差（Spread）**：买价和卖价之间的差额

示例：
```
AAPL 买价：$149.95
AAPL 卖价：$150.05
价差：$0.10
```

### 做市商

- 通过始终提供买卖报价来提供流动性
- 从价差中赚钱
- 确保你始终可以交易

### 成交量

- 一段时间内交易的股票数量
- 高成交量 = 更多流动性，更容易交易
- 低成交量 = 更难买卖，价差更大

## ✅ 答案

1. **交易 vs 投资**：交易是短期的，专注于价格波动；投资是长期的，专注于公司增长
2. **拥有股票**：你拥有公司的一小部分，有权获得利润（股息）和投票权
3. **为什么需要经纪商**：交易所只允许持牌经纪商交易；个人必须通过经纪商
4. **利润计算**：50股 × ($105 - $100) = $250利润
5. **价差**：买价和卖价之间的差额；重要是因为你在交易时会"损失"价差（以卖价买入，以买价卖出）
