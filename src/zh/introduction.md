# 交易学院

## 从零基础到专业交易员

欢迎来到**交易学院**，这是一个全面、结构化的学习路径，将带你从对交易一无所知到构建专业级的自动化交易系统。

### 🎯 本书适合谁

本书专为以下人群设计：
- **完全初学者** - 从未交易过的人
- **有抱负的算法交易员** - 想要自动化交易策略的人
- **开发者** - 对金融市场和量化交易感兴趣的人
- **任何人** - 认真系统学习交易的人

### 📚 你将学到什么

学完本书后，你将能够：

1. **理解交易基础** - 股票、市场、订单以及一切运作方式
2. **掌握技术分析** - 阅读图表、识别模式、使用指标
3. **构建交易策略** - 从简单到高级，从趋势跟踪到均值回归
4. **专业风险管理** - 仓位管理、止损、投资组合管理
5. **科学回测** - 在历史数据上测试策略，避免常见陷阱
6. **自动化交易** - 构建可以自动交易的生产级系统
7. **应用高级技术** - 机器学习、情绪分析、多资产策略

### 🛠️ 实践方法

这不仅仅是理论。每个模块都包括：

- **清晰的概念解释** 配合真实案例
- **代码示例** 使用 stock-agent-system
- **实践练习** 巩固学习
- **实战项目** 构建真实的交易系统
- **可视化** 直观理解概念

### 📖 本书如何组织

本书分为 **8 个渐进式模块**：

1. **交易基础** (2周) - 完全新手从这里开始
2. **技术分析基础** (2周) - 学习阅读图表
3. **技术指标** (2周) - 掌握交易员使用的工具
4. **交易策略** (3周) - 构建你自己的策略
5. **风险管理** (2周) - 保护你的资金
6. **回测与优化** (2周) - 科学测试策略
7. **自动化交易系统** (2周) - 构建生产系统
8. **高级主题** (3周) - 专业级技术

**预计总时间**：18周（4-5个月），每周10-15小时

### 🚀 开始学习

#### 前置要求

- 安装 Python 3.11+
- 基本的命令行知识
- 克隆并运行 [stock-agent-system](https://github.com/wjhou/stock-agent-system)
- 愿意每天学习和练习

#### 安装

```bash
# 克隆本仓库
git clone https://github.com/wjhou/trading-academy.git
cd trading-academy

# 安装依赖
pip install -r requirements.txt

# 开始阅读！
# 或者本地构建书籍：
mdbook serve --open
```

### 💡 学习理念

我们的方法基于经过验证的教育原则：

1. **渐进式复杂度** - 从简单开始，逐步构建
2. **主动学习** - 动手实践，而不仅仅是阅读
3. **立即应用** - 从第一天起就使用真实工具（stock-agent-system）
4. **间隔重复** - 概念在各模块中不断强化
5. **基于项目** - 构建真实系统，而非玩具示例

### ⚠️ 重要免责声明

- **仅供教育目的**：本书用于学习，不构成财务建议
- **先模拟交易**：在冒真实资金风险之前，务必先用模拟资金练习
- **风险警告**：交易涉及重大损失风险。永远不要用你承受不起损失的钱交易
- **无保证**：过去的表现不保证未来的结果
- **自行研究**：始终验证信息并进行自己的分析
