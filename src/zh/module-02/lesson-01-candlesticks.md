## 📖 什么是K线图？

**K线图**是价格走势的可视化表示，显示每个时间周期的四个关键数据点：开盘价、最高价、最低价和收盘价（OHLC）。

### 历史

- 由18世纪的日本米商开发
- 1990年代由史蒂夫·尼森引入西方市场
- 现在是交易者中最流行的图表类型

### 为什么选择K线图？

与折线图或柱状图相比，K线图：
- 一目了然地显示更多信息
- 揭示市场心理和情绪
- 使形态更容易识别
- 提供价格行为的视觉清晰度

## 🕯️ K线的解剖结构

每根K线都有几个组成部分：

```
     最高价（影线/上影线）
          |
    ┌─────────┐
    │         │  ← 实体（开盘价到收盘价）
    │  实体   │
    └─────────┘
          |
     最低价（影线/下影线）
```

### 组成部分

1. **实体**：开盘价和收盘价之间的矩形
   - **绿色/白色**：收盘价 > 开盘价（看涨）
   - **红色/黑色**：收盘价 < 开盘价（看跌）

2. **上影线**：从实体到最高价的线
   - 显示对更高价格的拒绝

3. **下影线**：从实体到最低价的线
   - 显示对更低价格的拒绝

### 示例

```
AAPL 日线K线（2026年3月4日）：
开盘价：$150.00
最高价：$153.50  ← 上影线到这里
收盘价：$152.00  ← 绿色实体顶部
最低价：$149.00  ← 下影线到这里

结果：看涨K线（收盘价高于开盘价）
```

## 📊 单K线形态

### 1. 十字星（Doji）

**外观**：实体非常小，开盘价 ≈ 收盘价

```
    |
    ─  ← 微小实体
    |
```

**含义**：犹豫不决，潜在反转
**背景**：在趋势极端点最为重要

**类型**：
- **长腿十字星**：两侧都有长影线（高波动性）
- **蜻蜓十字星**：长下影线，无上影线（看涨）
- **墓碑十字星**：长上影线，无下影线（看跌）

### 2. 锤子线（Hammer）

**外观**：顶部小实体，长下影线（2倍以上实体大小）

```
    ┌──┐
    └──┘
      |
      |
      |
```

**含义**：下跌趋势后的看涨反转
**心理**：卖方将价格压低，但买方重新获得控制权

**要求**：
- 出现在下跌趋势之后
- 下影线至少是实体长度的2倍
- 几乎没有或没有上影线
- 实体颜色不太重要（绿色更佳）

### 3. 倒锤子线（Inverted Hammer）

**外观**：底部小实体，长上影线

```
      |
      |
      |
    ┌──┐
    └──┘
```

**含义**：潜在看涨反转
**心理**：买方试图推高，失败了，但显示出买入兴趣

### 4. 流星线（Shooting Star）

**外观**：底部小实体，长上影线（倒锤子线的看跌版本）

```
      |
      |
      |
    ┌──┐
    └──┘
```

**含义**：上涨趋势后的看跌反转
**心理**：多头推高，空头接管控制权

**要求**：
- 出现在上涨趋势之后
- 上影线至少是实体长度的2倍
- 几乎没有或没有下影线

### 5. 上吊线（Hanging Man）

**外观**：顶部小实体，长下影线（锤子线的看跌版本）

```
    ┌──┐
    └──┘
      |
      |
      |
```

**含义**：上涨趋势后的看跌反转
**心理**：尽管有买入压力，卖方正在出现

### 6. 光头光脚线（Marubozu）

**外观**：长实体，无影线（或非常小）

```
看涨：          看跌：
┌────────┐      ┌────────┐
│        │      │        │
│        │      │        │
│        │      │        │
└────────┘      └────────┘
（绿色）         （红色）
```

**含义**：一个方向的强烈信念
**看涨光头光脚线**：强劲买入，无拒绝
**看跌光头光脚线**：强劲卖出，无拒绝

### 7. 陀螺线（Spinning Top）

**外观**：小实体，两侧都有长影线

```
      |
    ┌──┐
    └──┘
      |
```

**含义**：犹豫不决，买方和卖方之间的平衡
**背景**：可能表明趋势减弱

## 🔄 多K线组合形态

### 1. 吞没形态（Engulfing Pattern）

**看涨吞没**：
```
第1天    第2天
 ┌─┐    ┌─────┐
 └─┘    │     │
（红色） │     │
        └─────┘
        （绿色）
```

**要求**：
- 形态前有下跌趋势
- 第2天实体完全吞没第1天实体
- 第2天开盘更低，收盘高于第1天

**含义**：强烈反转信号，多头压倒空头

**看跌吞没**：相反（绿色然后红色）

### 2. 刺透形态（Piercing Pattern）

```
第1天      第2天
┌─────┐    ┌────┐
│     │    │    │
│     │  ┌─┘    │
└─────┘  │      │
（红色）  └──────┘
         （绿色）
```

**要求**：
- 形态前有下跌趋势
- 第2天开盘低于第1天最低价
- 第2天收盘高于第1天实体的50%

**含义**：看涨反转，买方介入

### 3. 乌云盖顶（Dark Cloud Cover）

```
第1天      第2天
┌──────┐   ┌────┐
│      │   │    │
│      │   │    └─┐
└──────┘   │      │
（绿色）    └──────┘
           （红色）
```

**要求**：
- 形态前有上涨趋势
- 第2天开盘高于第1天最高价
- 第2天收盘低于第1天实体的50%

**含义**：看跌反转，卖方接管控制权

### 4. 早晨之星（Morning Star）（看涨）

```
第1天    第2天   第3天
┌────┐    ─     ┌────┐
│    │   ─ ─    │    │
│    │    ─     │    │
└────┘          └────┘
（红色）（十字星）（绿色）
```

**要求**：
- 形态前有下跌趋势
- 第1天：大型看跌K线
- 第2天：小实体（十字星/陀螺线），向下跳空
- 第3天：大型看涨K线，收盘高于第1天中点

**含义**：强烈看涨反转

### 5. 黄昏之星（Evening Star）（看跌）

早晨之星的相反 - 表示上涨趋势后的看跌反转

### 6. 三白兵（Three White Soldiers）

```
第1天   第2天   第3天
┌───┐   ┌───┐   ┌───┐
│   │   │   │   │   │
│   │   │   │   │   │
└───┘   └───┘   └───┘
（绿色）（绿色）（绿色）
```

**要求**：
- 三根连续的看涨K线
- 每根在前一根实体内开盘
- 每根在其最高价附近收盘
- 稳步向上推进

**含义**：强烈看涨延续

### 7. 三只乌鸦（Three Black Crows）

三白兵的相反 - 看跌延续

## 💻 实际应用

### 在交易中使用K线

1. **识别趋势**
   - 形态在不同背景下有不同含义
   - 反转形态需要一个趋势来反转

2. **寻找确认**
   - 不要仅凭形态交易
   - 等待下一根K线确认
   - 与成交量和其他指标一起使用

3. **考虑时间框架**
   - 日线K线：更可靠，信号较慢
   - 小时K线：信号更快，噪音更多
   - 5分钟K线：非常嘈杂，可靠性较低

### 交易设置示例

```python
# 检测锤子线形态的伪代码
def is_hammer(candle):
    body = abs(candle.close - candle.open)
    lower_wick = min(candle.open, candle.close) - candle.low
    upper_wick = candle.high - max(candle.open, candle.close)

    # 锤子线要求
    if lower_wick >= 2 * body:  # 长下影线
        if upper_wick <= 0.1 * body:  # 小上影线
            if in_downtrend():  # 在下跌趋势之后
                return True
    return False

# 等待确认
if is_hammer(yesterday) and today.close > yesterday.high:
    # 考虑买入
    enter_long_position()
```

## ✅ 答案

1. **四个价格点**：开盘价、最高价、最低价、收盘价（OHLC）
2. **长下影线**：卖方将价格压低，但买方拒绝更低的价格并推回
3. **锤子线 vs 上吊线**：外观相同（小实体，长下影线），但锤子线出现在下跌趋势之后（看涨），上吊线出现在上涨趋势之后（看跌）
4. **极端点的十字星**：在强劲走势后显示犹豫不决，表明趋势可能已经耗尽并准备反转
5. **吞没形态确认**：等待下一根K线朝形态方向收盘，检查成交量增加，寻找附近支撑/阻力的突破
