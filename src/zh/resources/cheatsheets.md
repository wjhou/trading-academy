# 交易速查表

常见交易概念的快速参考指南。

## 订单类型速查表

| 订单类型 | 何时使用 | 优点 | 缺点 |
|---------|---------|------|------|
| **市价单** | 需要立即执行 | 快速、保证成交 | 无价格控制、滑点 |
| **限价单** | 想要特定价格 | 价格控制 | 可能不成交 |
| **止损单** | 防止损失 | 自动风险管理 | 可能在下跌时触发 |
| **止损限价单** | 触发后的价格控制 | 更多控制 | 可能不成交 |

## K线形态速查表

### 看涨形态（价格可能上涨）

| 形态 | 视觉特征 | 含义 |
|-----|---------|------|
| **锤子线** | 长下影线，小实体在顶部 | 拒绝低点，买家介入 |

| **三白兵** | 3根连续的长阳线 | 强劲上升趋势 |

|-----|---------|------|
| **流星线** | 长上影线，小实体在底部 | 拒绝高点，卖家介入 |
| **看跌吞没** | 大阴线吞没前一根阳线 | 强劲卖压 |

## 市场时间速查表

### 美国市场（东部时间）

| 时段 | 时间 | 特点 |
|-----|------|------|
| **盘前** | 4:00 AM - 9:30 AM | 低成交量、宽价差 |
| **开盘** | 9:30 AM - 10:30 AM | 高波动率、最大成交量 |
| **午盘** | 10:30 AM - 3:00 PM | 较低成交量、盘整 |
| **尾盘** | 3:00 PM - 4:00 PM | 高成交量、趋势延续 |
| **盘后** | 4:00 PM - 8:00 PM | 低成交量、宽价差 |

### 最佳交易时间

- **9:30-11:00 AM**：最高成交量和波动率
- **3:00-4:00 PM**：第二好的时间，收盘走势
- **避免**：前15分钟（对初学者来说太波动）
- **避免**：12:00-2:00 PM（午餐时间，低成交量）

## 交易心理速查表

### 要避免的情绪

| 情绪 | 症状 | 解决方案 |
|-----|------|---------|
| **恐惧** | 不敢入场交易 | 遵循计划，从小开始 |
| **贪婪** | 仓位过大 | 坚持仓位规模规则 |
| **FOMO** | 追涨 | 等待设置，保持耐心 |
| **报复** | 亏损后交易 | 休息一下，回顾错误 |
| **过度自信** | 忽视风险 | 记住损失会发生 |

### 交易规则

1. **每笔交易风险不超过1-2%**
2. **始终使用止损**
3. **让赢家奔跑，快速止损**
4. **情绪化时不要交易**
5. **保持交易日志**
6. **每周回顾交易**
7. **新策略先模拟交易**
8. **永远不要用输不起的钱交易**

## 快速公式

### 风险收益比
```
风险 = 入场价格 - 止损
收益 = 止盈 - 入场价格
比率 = 收益 ÷ 风险

示例：
入场：$100，止损：$95，目标：$110
风险：$5，收益：$10
比率：2:1（好！）
```

### 仓位规模
```
仓位规模 = 风险金额 ÷ (入场 - 止损)

示例：
账户：$10,000
风险：1% = $100
入场：$50，止损：$48
仓位规模 = $100 ÷ $2 = 50股
```

### 盈亏百分比
```
盈亏% = ((出场 - 入场) ÷ 入场) × 100

示例：
入场：$100，出场：$110
盈亏% = ($10 ÷ $100) × 100 = 10%
```

### 夏普比率（风险调整后收益）
```
夏普 = (平均收益 - 无风险利率) ÷ 标准差

越高越好（>1好，>2优秀）
```
