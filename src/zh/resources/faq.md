# 常见问题解答 (FAQ)

关于交易和本学院的常见问题。

## 🎓 关于交易学院

### 问：这个课程真的免费吗？
**答**：是的！交易学院完全免费且开源。所有内容都在 GitHub 上以 MIT 许可证提供。

### 问：完成课程需要多长时间？
**答**：完整课程设计为 18 周（4-5 个月），每周 10-15 小时。但你可以按照自己的节奏学习。

### 问：我需要编程经验吗？
**答**：基础的 Python 知识会有帮助，但我们会教你所需的内容。如果你完全没有编程经验，建议先花 1-2 周学习 Python 基础。

### 问：我可以跳过某些模块吗？
**答**：我们不建议这样做。每个模块都建立在前面模块的基础上。但是，如果你已经掌握了基础知识，可以通过完成模块练习来测试自己。

## 💰 关于交易

### 问：开始交易需要多少资金？
**答**：模拟交易（练习）：$0
真实交易：建议最少 $1,000-5,000。永远不要用你输不起的钱交易。

### 问：我真的能通过交易赚钱吗？
**答**：可以，但很困难。大多数初学者都会亏钱。成功需要教育、练习、纪律和风险管理。本学院教你方法，但不保证结果。

### 问：我能通过交易赚多少钱？
**答**：现实预期：
- 初学者：每年 -20% 到 +10%（学习阶段）
- 中级：每年 10-30%
- 高级：每年 30-100%+
- 专业：差异很大

**警告**：任何承诺保证回报的人都在撒谎。

### 问：我应该辞职专职交易吗？
**答**：不！至少要等到你持续盈利 2-3 年。保留你的日常工作，最初兼职交易。
